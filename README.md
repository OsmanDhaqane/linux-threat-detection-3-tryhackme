# Linux Threat Detection 3 - TryHackMe

Walkthrough for the TryHackMe **Linux Threat Detection 3** room.

This room focuses on detecting the later stages of Linux attacks through system logs and auditd. It covers reverse shells, privilege escalation, startup persistence, and account persistence.

## Topics Covered

- Reverse shell detection
- Command injection through TryPingMe
- Auditd log investigation
- Privilege escalation detection
- Cron persistence
- Systemd service persistence
- New sudo user detection
- SSH key persistence
